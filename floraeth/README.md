🚀 **Live Deployment:** [https://floraeth.netlify.app/](https://floraeth.netlify.app/)

A premium, fully responsive, single-page landing website for a luxury garden design and floral boutique brand called "Floraeth". It features cinematic video backgrounds, custom Framer Motion animations, and a bespoke dark botanical aesthetic.

## ✨ Key Features
- **Cinematic Video Hero:** Full-viewport looping video background with glowing typography.
- **Scroll-Driven Animations:** Parallax images and a traveling glowing timeline dot driven by scroll position.
- **Custom Liquid Glass:** A unique warm-tinted botanical glassmorphism effect.
- **Dynamic Bento Grid & Carousels:** Interactive service cards and a horizontal image gallery carousel.

## 🛠️ Tech Stack
- **Framework:** React + Vite + TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## 🚀 Getting Started
1. Clone the repository
2. Navigate to `/floraeth`
3. Run `npm install`
4. Run `npm run dev`

## 📂 Project Structure
- `src/` - React components, pages, and CSS
- `src/assets/` - Videos and images
- `public/` - Static assets

## 🎨 Design System
- **Colors:** Deep forest black-green, warm cream white, living moss green, and warm terracotta gold.
- **Typography:** Playfair Display (Headings), Lora (Editorial/Quotes), DM Sans (UI Text).
