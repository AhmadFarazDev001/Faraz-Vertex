## ✦ The Archipelagos (Projects)

This repository serves as a centralized hub for multiple independent web projects, each housed within its own isolated directory. 

### 1. [AETHEL](./aethel)
**Ultra-Luxury Shoe Atelier Landing Page**
An editorial, single-page web experience built for an ultra-luxury bespoke footwear brand. 
- **Tech Stack:** React, Vite, Tailwind CSS, TypeScript.
- **Key Features:**
  - **Cinematic Aesthetics:** Custom SVG `feTurbulence` noise overlays simulating raw film grain, paired with a dark obsidian and soft champagne color palette.
  - **Hardware-Accelerated Animations:** Buttery smooth, CSS-driven split-screen interactions replacing heavy JavaScript layout thrashing.
  - **Responsive Architecture:** Built using modern CSS Grid and Flexbox for seamless flow between desktop and mobile viewports.
  - **Fluid Typography:** Implementation of CSS `clamp()` functions for massive, natively responsive headline text.

## 🛠️ Global Tech Stack
Across the various projects in this repository, the primary technologies utilized include:
